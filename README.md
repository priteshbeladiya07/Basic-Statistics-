# 📊 Statistics with Python

This repository contains my hands-on learning and implementation of **Statistics using Python** through Jupyter Notebooks. It covers descriptive statistics, data preprocessing, hypothesis testing, correlation analysis, regression, and ANOVA with practical examples.

---

## 📚 Topics Covered

### 📈 Descriptive Statistics
- Mean
- Median
- Mode
- Variance
- Standard Deviation
- Skewness
- Kurtosis

### 🧹 Data Preprocessing
- Handling Missing Values
- Removing Outliers
  - IQR Method
  - Z-Score Method

### 🧪 Hypothesis Testing
- Z-Test
- One Sample T-Test
- Independent Two Sample T-Test
- Paired T-Test
- Proportion (P) Test
- P-Value Interpretation
- Confidence Intervals

### 📊 Statistical Analysis
- Correlation Analysis
- Pearson Correlation
- Spearman Correlation
- Covariance

### 📉 Regression Analysis
- Simple Linear Regression
- Multiple Linear Regression
- Regression Visualization

### 📋 ANOVA
- One-Way ANOVA
- Interpreting ANOVA Results

### 🔢 Chi-Square Test
- Chi-Square Test of Independence
- Contingency Tables
- Interpretation of Results

### 📌 Additional Concepts
- Probability Basics
- Normal Distribution
- Sampling Techniques
- Central Limit Theorem
- Statistical Decision Making
